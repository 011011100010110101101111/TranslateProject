2019年面向所有人的最佳 Linux 发行版
======

_**简介:哪个是最好的 Linux 发行版呢？这个问题是没有明确的答案的。这就是为什么我们在各种分类中汇编了这个最佳Linux列表的原因。**_

有许多 Linux 发行版。我甚至想不出一个确切的数量，因为你会发现很多不同的 Linux 发行版。

其中有些只是彼此的复制品，而有些往往是独一无二的。这虽然有点混乱——但这也是 Linux 的优点。

不用担心，尽管有成千上万的发行版，在这篇文章中，我已经列出了目前最好的 Linux 发行版。当然，这个列表是主观的。但是，在这里，我们试图对发行版进行分类——所以每个发行版本都有自己的特点的。

  * [面向初学者的 Linux 用户的最佳发行版][1]
  * [最佳服务器 Linux 发行版][2]
  * [可以在旧计算机上运行的最佳 Linux 发行版][3]
  * [面向高级 Linux 用户的最佳发行版][4]
  * [最佳常青树 Linux 发行版][5]



**注:** _该列表没有特定的排名顺序。_

### 面向初学者的最佳 Linux 发行版 

在这个分类中，我们的目标是列出开箱即用的发行版。你不需要深度学习，你可以在安装后马上开始使用，不需要知道任何命令或提示。

#### Ubuntu

![][6]

Ubuntu 无疑是最流行的 Linux 发行版之一。你甚至可以在很多笔记本电脑上找到它的预装版本。

用户界面很容易适应。如果你愿意，你可以根据自己的要求轻松定制它的外观。无论哪种情况，你都可以选择安装一个主题。在开始之前，你可以先了解更多关于[如何在Ubuntu安装主题的][7]的信息。

除了它本身提供的内容外，你会发现一个巨大的 Ubuntu 用户在线社区。因此，如果你有问题——可以去任何论坛(或子论坛)寻求帮助。如果你正在寻找解决方案，你应该看看我们对[Ubuntu][8] 的报道(我们有很多关于 Ubuntu 的教程和建议)。

[Ubuntu][9]

#### Linux Mint

![Linux Mint 19 Cinnamon desktop screenshot][10]

Linux Mint Cinnamon 是另一个受初学者欢迎的 Linux 发行版。默认的 Cinnamon 桌面类似于 Windows XP，这就是为什么当 Windows XP 停止维护时许多用户选择它的原因。

Linux Mint 基于 Ubuntu，因此它拥有 Ubuntu 可用的所有应用程序。简单易用是它成为新的 Linux 用户的突出选择的原因。

[Linux Mint][11]

#### elementary OS

![][12]

elementary OS 是我用过的最漂亮的 Linux 发行版之一。用户界面类似于苹果操作系统——所以如果你已经使用了苹果系统，它会很容易适应你。 

该发行版基于 Ubuntu，重点是提供一个用户友好的 Linux 环境，该环境在考虑性能的同时尽可能美观。如果你选择安装 elementary OS，[在安装elementary OS 后要做的11件事的清单][13] 会派上用场。

[elementary OS][14]

#### MX Linux

![][15]

大约一年前，MX Linux 成为众人瞩目的焦点。现在(在发表这篇文章的时候)，它是最受欢迎的 Linux 发行版在[DistroWatch.com][16]上。如果你还没有使用它——当你开始使用它时，你会感到惊讶。

与 Ubuntu 不同，MX Linux是基于 Debian 的[滚动发行版][17]，Xfce 作为其桌面环境。除了无可挑剔的稳定性之外，它还配备了许多图形用户界面工具，这使得最初对 Windows/Mac 感到舒适的用户更加容易使用。

此外，软件包管理器非常适合一键安装。你甚至可以搜索[Flatpak][18]软件包，并立即安装它(默认情况下，Flathub 在软件包管理器中是可用的来源之一)。

[MX Linux][19]

#### Zorin OS

![][20]

Zorin OS 是另一个基于 Ubuntu 的发行版，它恰好是桌面上最漂亮、最直观的操作系统之一。尤其是在[Zorin OS 15 release][21]之后——我绝对会推荐给没有任何 Linux 经验的用户。许多基于图形用户界面的应用程序也应运而生。

您也可以将其安装在旧电脑上，但是，请确保选择“精简”版本。此外，你还有“核心”、“教育”和 “终极”版本。你可以选择免费安装核心版，但是如果你想支持开发人员并帮助改进 Zorin，请考虑获得终极版。

Zorin OS 是由爱尔兰的两个青少年创建的。你可以[在这里读他们的故事][22]。

[Zorin OS][23]

**其他选项**

[Deepin][24]和其他的Ubuntu(如 Kubuntu、Xubuntu)也是初学者的首选。如果你想找寻更多的选择，你可以看看。 

如果你想要挑战自己，你可以试试 Fedora——但是一定要看看我们关于[Ubuntu vs Fedora][25]的文章，从桌面的角度做出更好的选择。 

### 最好的服务器发行版

对于服务器来说，选择 Linux 发行版归结为稳定性、性能和企业支持。如果你只是在试验，你可以尝试任何你想要的发行版。

但是，如果你要为 web server 或任何重要的东西安装它，你应该看看我们的一些建议。 

#### Ubuntu 服务器

根据您的需要，Ubuntu 为你的服务器提供了不同的选项。如果您正在寻找运行在 AWS、Azure、谷歌云平台等平台上的优化解决方案。，[Ubuntu 云][26]是一个很好的选择。

无论是哪种情况，你都可以选择 Ubuntu 服务器包，并将其安装在你的服务器上。然而，Ubuntu 在云上部署时也是最受欢迎的Linux 发行版(根据数字——[来源1][27]，[来源2][28])。 

请注意，我们建议你选择LTS版，除非你有特殊要求。

[Ubuntu Server][29]

#### Red Hat 企业版 Linux

Red Hat 企业 Linux 是面向企业和组织的顶级 Linux 平台。如果我们按数字来看，Red Hat可能不是服务器领域最受欢迎的。但是，有相当一部分企业用户依赖 RHEL (比如联想)。

从技术上讲，Fedora 和 Red Hat是相关联的。无论Red Hat 支持什么——在RHEL上市之前，都要在 Fedora 上进行测试。我不是定制需求的服务器分发专家，所以你一定要查看他们在[官方文档][30]以了解它是否适合你。 

[Red Hat Enterprise Linux][31]

#### SUSE Linux 企业服务器

![Suse Linux Enterprise \(Image: Softpedia\)][32]

别担心，不要把这和 OpenSUSE 混淆。所有的东西都归入一个共同的品牌 “SUSE” ——但是 OpenSUSE 是一个开源发行版，目标是社区，并且由社区维护。

SUSE Linux 企业服务器是基于云的服务器最受欢迎的解决方案之一。为了获得管理开源解决方案的优先支持和帮助，您必须选择订阅。

[SUSE Linux Enterprise Server][33]

#### CentOS

![][34]

正如我提到的，你需要订阅 RHEL。但是，CentOS 更像是 RHEL 的社区版，因为它是从Red Hat 企业 Linux 派生出来的。而且，它是开源的，也是免费的。尽管与过去几年相比，使用 CentOS 的主机提供商数量明显减少，但这仍然是一个很好的选择。

CentOS 可能没有加载最新的软件包，但它被认为是最稳定的发行版之一,你可以在各种云平台上找到 CentOS 镜像。如果没有，您可以选择 CentOS 提供的自托管镜像。

[CentOS][35]

### 旧电脑的最佳 Linux 发行版

如果你有一台旧电脑，或者你真的不需要升级你的系统，你仍然可以尝试一些最好的Linux 发行版。

我们已经详细讨论了一些[最好的轻量级Linux发行版][42]。在这里，我们将只提到那些真正突出的东西(以及一些新的补充)。

#### Puppy Linux

![][43]

Puppy Linux 实际上是最小的发行版本之一。当我第一次开始使用 Linux 时，我的朋友建议我尝试一下 Puppy Linux，因为它可以轻松地在旧的硬件配置上运行。  

如果你想在你的旧电脑上享受一次爽快的体验，那就值得去看看。多年来，随着几个新的有用特性的增加，用户体验得到了改善。
[Puppy Linux][44]

#### Solus Budgie

![][45]

在最近的一个主要版本——[Solus 4 Fortitude][46]之后，它是一个令人印象深刻的轻量级桌面操作系统。你可以选择像 GNOME 或 MATE 这样的桌面环境。然而，Solus Budgie 恰好是我的最爱之一，它是一款适合初学者的成熟的Linux发行版，同时对系统资源要求很少。
[Solus][47]

#### Bodhi

![][48]

Bodhi Linux 是建立在 Ubuntu 之上的。然而，与Ubuntu不同，它在旧配置上运行良好。

这个发行版的主要亮点是它的[Moksha 桌面][49](这是 Enlightenment 17桌面的延续)。用户体验直观且尖叫迅速。即使它不是我个人使用的东西，你也应该在你的旧系统上试一试。

[Bodhi Linux][50]

#### antiX

![][51]

antiX——它也部分负责 MX Linux，是一个轻量级的 Linux 发行版，为新旧计算机量身定制。用户界面并不令人印象深刻——但它可以像预期的那样工作。 

它基于 Debian，可以作为一个在线光盘发行版使用，而不需要安装它。antiX还提供在线引导加载程序。与其他发行版相比，您可以保存设置，这样就不会在每次重新启动时丢失设置。不仅如此，您还可以通过其“在线持久性”功能将更改保存到根目录中。 


因此，如果你正在寻找一个在线 USB 发行版，在旧硬件上提供快速的用户体验——antiX是一个不错的选择。 

[antiX][52]

#### Sparky Linux

![][53]

Sparky Linux  是基于Debian的，它是低端系统的完美的 Linux 发行版。伴随着令人尖叫的快速体验，Sparky Linux 为不同的用户提供了几个特殊版本(或变种)。 

例如，它提供了针对一组用户的稳定版本(有各种版本)和滚动版本。Sparky Linux GameOver版非常受玩家欢迎，因为它包含了一堆预装的游戏。你可以查看我们的[最佳Linux游戏发行版][54]–如果你也想在你的系统上玩游戏。

#### 其他选择

您也可以尝试[Linux精简版][55]，[Lubuntu][56]，[Peppermint][57]作为一些轻量级Linux发行版。 

### 面向高级用户的最佳 Linux 发行版

一旦你习惯了各种软件包管理器和命令来帮助你解决任何问题，你就可以开始找寻只为高级用户定制的 Linux 发行版。

当然，如果你是专业人士，你会有一套具体的要求。然而，如果你已经作为普通用户使用了一段时间——以下发行版值得一试。

#### Arch Linux

![Image Credits: Samiuvic / Deviantart][58]

Arch Linux 本身是一个简单而强大的发行版，具有巨大的学习曲线。不像其系统，你不会一次就把所有东西都预先安装好。你必须配置系统并根据需要添加软件包。 

此外，在安装 Arch Linux时，必须遵循一组命令(没有图形用户界面)。要了解更多信息，你可以按照我们关于[如何安装Arch Linux][59]的指南进行操作。如果你要安装它，你还应该知道在[安装Arch Linux 后需要做的一些基本事情][60]。这会帮助你有一个跳跃的开始。

除了所有的通用性和简单性之外，值得一提的是 Arch Linux 背后的社区非常活跃。所以，如果你遇到问题，你不用担心。

[Arch Linux][61]

#### Gentoo

![Gentoo Linux][62]

如果你知道如何编译源代码，Gentoo Linux是你必须尝试的版本。这也是一个轻量级的发行版，但是，您需要具备必要的技术知识才能使它发挥作用。

当然，[官方手册》[63]提供了许多你需要知道的信息。但是，如果你不确定自己在做什么——你需要花很多时间去想如何充分利用它。

[Gentoo Linux][64]

#### Slackware

![Image Credits: thundercr0w / Deviantart][65]

Slackware 是仍然重要的最古老的 Linux 发行版之一。如果你愿意编译或开发软件来为自己建立一个完美的环境—— Slackware 是一个不错的选择。

如果你对一些最古老的linux发行版感到好奇，我们有一篇关于[最早的Linux发行版][66]——可以去看看。

尽管使用它的用户/开发人员的数量已经显著减少，但对于高级用户来说，它仍然是一个极好的选择。此外，随着[Slackware 获得 Patreon 页面][67]的最新消息，我们希望Slackware 继续作为最好的 Linux 发行版之一存在。

[Slackware][68]

### 最佳多用途Linux发行版

有些 Linux 发行版既可以作为初学者友好的桌面又可以作为高级操作系统的服务器。因此，我们考虑为这样的发行版编辑一个单独的部分。

如果你不同意我们的观点(或者有建议要补充)，请在评论中告诉我们。以下是我们认为对每个用户都有用的东西:

#### Fedora

![][69]

Fedora 提供两个独立的版本——一个用于台式机/笔记本电脑，另一个用于服务器(Fedora工作站和Fedora服务器)。

因此，如果你正在寻找一款时髦的桌面操作系统——既有潜在的学习曲线，又对用户友好——Fedora是一个选择。无论是哪种情况，如果你正在为你的服务器寻找一个Linux操作系统——这也是一个不错的选择。

[Fedora][70]

#### Manjaro

![][71]

Manjaro[Arch Linux][72]。不用担心，虽然 Arch Linux是为高级用户量身定制的，但Manjaro 让新手更容易上手。这是一个简单且对初学者友好的 Linux 发行版。用户界面足够好，并且内置了一系列有用的图形用户界面应用程序。

下载时，您可以选择 Manjaro 的[桌面环境][73]。就个人而言，我喜欢 Manjaro 的KDE桌面。

[Manjaro Linux][74]

### Debian

![Image Credits: mrneilypops / Deviantart][75]

嗯，Ubuntu 是基于 Debian的——所以它本身是一个非常好的发行版本。Debian是台式机和服务器的理想选择。

这可能不是对初学者最友好型操作系统——但你可以通过[官方文档][76]轻松开始。[Debian 10 Buster][77]最近的发行引入了许多变化和必要的改进。所以，你必须试一试！

**收尾**

总的来说，这些是我们推荐你去尝试的最好的 Linux 发行版。是的，还有许多其他的Linux发行版值得一提——但是根据个人喜好，对每个发行版来说——选择将是主观的。


但是，我们也为[Windows users][78]、[黑客和笔友][41]、[玩家][54]、[程序员][39]和[隐私爱好者]提供了一个独立的发行版列表][79]所以，如果你感兴趣的话——一定要去看看。

如果你认为我们遗漏了你最喜欢的一个，值得作为最好的Linux发行版之一，请在下面的评论中告诉我们你的想法，我们将更新这篇文章。
